# 复利计算器

一个简洁高效的在线复利计算器，支持多种复利频率，帮助您快速计算投资收益。

## ✨ 功能特点

- 🎯 **双模式计算**：支持正向计算和目标金额计算两种模式
- 📊 **多种频率**：支持年、半年、季度、月、日复利计算
- 🔍 **详细分析**：显示最终金额、利息总额、投资回报率和复利效应
- 📱 **响应式设计**：完美适配手机、平板和电脑
- ⌨️ **键盘支持**：支持回车键快速计算，Ctrl+Enter快捷计算
- 🎨 **现代化UI**：采用 Noto Sans SC 字体，美观易读
- ⚡ **实时验证**：输入时即时验证，提供友好提示
- 📈 **智能显示**：大额数字自动显示为"万"或"M"单位

## 🚀 使用方法

### 正向计算模式
1. **输入本金**：初始投资金额（元）
2. **输入年利率**：预期年化收益率（%）
3. **输入年限**：投资期限（年）
4. **选择频率**：复利计算频率
5. **点击计算**：查看最终金额和收益

### 目标金额模式
1. **选择模式**：点击"目标金额"切换到目标模式
2. **输入目标金额**：您希望达到的最终金额
3. **输入年利率**：预期年化收益率（%）
4. **输入年限**：计划投资期限（年）
5. **选择频率**：复利计算频率
6. **点击计算**：查看需要投入的本金

## 📊 计算结果说明

### 正向计算结果
- **最终金额**：包含本金和利息的总金额
- **获得利息**：投资期间获得的利息总额
- **投资回报率**：相对于本金的收益率
- **复利效应**：相对于单利的额外收益比例

### 目标金额计算结果
- **所需本金**：需要投入的初始金额
- **目标金额**：您希望达到的金额
- **总利息**：投资期间获得的利息
- **投资建议**：包含收益率、年限、复利频率的详细信息

## 🧮 复利计算公式

### 正向计算
```
A = P(1 + r/n)^(nt)
```

### 目标金额计算（反向计算）
```
P = A / (1 + r/n)^(nt)
```

其中：
- A = 最终金额/目标金额
- P = 本金/所需本金
- r = 年利率（小数形式）
- n = 每年复利次数
- t = 年数
